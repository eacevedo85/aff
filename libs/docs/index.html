Header

CSS

Usar la libreria LIB_JS para poder agregar sus scripts en el controlador

<?php 
$header[ 'css' ] = TRUE;
$header[ 'lib_css' ] = array(
	'<link href=" ' . base_url() . 'libs/css/style.css" rel="stylesheet" ' ); 
?>
$this->load->view( 'header', $header );

Footer

Lib JS

Usar la libreria LIB_JS para poder agregar sus scripts en el controlador

<?php 
$footer[ 'libs' ] = TRUE;
$footer[ 'lib_js' ] = array(
	'<script src=" ' . base_url() . 'libs/js/jquery.validate.js"></script>' ); 
?>

PHP

Agrega vistas nuevas en PHP, su uso puede servir para generar archivos de Javascript con contenido que ejecute PHP

<?php 
$footer[ 'php' ] = TRUE;
$footer[ 'lib_php' ][] = 'js/usuarios.js.php';
?>

Debug

Activar el debug

<?php 
$footer[ 'debug' ] = TRUE;
$this->load->view( 'footer', $footer );
?>
